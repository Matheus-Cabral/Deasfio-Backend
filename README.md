# Desafio Backend

Implementação dos três desafios propostos:

1. **Cálculo de comissão de vendedores** a partir de um arquivo JSON de vendas.
2. **Sistema simples de movimentação de estoque**, com lançamento de entradas/saídas e retorno do saldo final do produto.
3. **Cálculo de juros por atraso**, considerando multa de 2,5% ao dia entre a data de vencimento e a data atual.

## Tecnologias

- .NET 9
- C#
- Console Application
- System.Text.Json para leitura dos dados em JSON

## Como executar

```bash
dotnet run --project DesafioBackend
